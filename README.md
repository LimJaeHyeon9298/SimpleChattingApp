## :coffee: 소감

Firebase를 이용한 유저 등록, 로그인, 로그인 후 메세지 저장 및 변화 감지 등 전반적인 기능들을 사용해서 채팅앱을 구성해봤다. 다음에 개인 앱을 만들때 Firebase 기능을 이용해서 유저관리, 로그인 등을 손 쉽게 구현 할 수 있다라는 자신감이 생겼다.
